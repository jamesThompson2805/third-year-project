\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename I$>$\newline
class pgbar\+::scope\+::\+Iter\+Span$<$ I $>$}
An undirectional range delimited by a pair of iterators, including pointer types.

When the type of iterator is pointer, it can figure out whether the iterator is reversed, and iterate it normally.

Accepted iterator types must satisfy subtractable.
