\item {\bfseries{Header-\/only}}\+: 所有功能都包含在 {\ttfamily include/pgbar} 下的单个 {\ttfamily .hpp} 文件中。
\item {\bfseries{低运行时开销}}\+: 更新进度条的开销成本极低。
\item {\bfseries{Unicode 支持}}\+: 按 UTF-\/8 编码解析每个字符串。
\item {\bfseries{RGB 支持}}\+: 可通过十六进制 RGB 值定制进度条颜色。
\item {\bfseries{可选的线程安全}}\+: 可以通过模板参数调节进度条的线程安全性。
\item {\bfseries{类 {\ttfamily tqdm} 接口}}\+: 利用模板技巧提供了友好的使用方式。
\item {\bfseries{Modern C++ 规范}}\+: （基本上）遵循 Modern C++ 的最佳实践。
