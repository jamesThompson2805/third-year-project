\doxysubsection{Detailed Description}
A wrapper that stores ordered units for information rate formatting (e.\+g. B/s, k\+B/s).

Encapsulates four consecutive scaling units where each unit is scaled by the configured magnitude factor (default 1,000x if no {\ttfamily \doxylink{structpgbar_1_1option_1_1Magnitude}{option\+::\+Magnitude}} is explicitly set).

Unit order MUST be ascending\+: \mbox{[}base\+\_\+unit, scaled\+\_\+unit\+\_\+1, scaled\+\_\+unit\+\_\+2, scaled\+\_\+unit\+\_\+3\mbox{]}.

Example\+:


\begin{DoxyItemize}
\item magnitude=1000\+: \mbox{[}"{}\+B/s"{}, "{}k\+B/s"{}, "{}\+MB/s"{}, "{}\+GB/s"{}\mbox{]}
\item magnitude=1024\+: \mbox{[}"{}\+B/s"{}, "{}\+Ki\+B/s"{}, "{}\+Mi\+B/s"{}, "{}\+Gi\+B/s"{}\mbox{]}
\end{DoxyItemize}

Scaling logic\+: value \texorpdfstring{$>$}{>}= magnitude -\/\texorpdfstring{$>$}{>} upgrade to next unit tier.
