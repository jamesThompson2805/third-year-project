\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename N$>$\newline
class pgbar\+::scope\+::\+Numeric\+Span$<$ N $>$}
An undirectional range delimited by an numeric interval \mbox{[}start, end).

The {\ttfamily end} can be less than the {\ttfamily start} only if the {\ttfamily step} is negative, otherwise it throws exception {\ttfamily \doxylink{classpgbar_1_1exception_1_1InvalidArgument}{pgbar\+::exception\+::\+Invalid\+Argument}}.
