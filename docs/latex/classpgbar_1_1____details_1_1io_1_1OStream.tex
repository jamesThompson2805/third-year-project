\doxysubsection{Detailed Description}
\subsubsection*{template$<$Channel Stream\+Type$>$\newline
class pgbar\+::\+\_\+\+\_\+details\+::io\+::\+OStream$<$ Stream\+Type $>$}
A helper output stream that writes the data to {\ttfamily stdout} or {\ttfamily stderr} directly.

It holds a proprietary buffer so that don\textquotesingle{}t have to use the common output buffers in the standard library.

If the local platform is neither {\ttfamily Windows} nor {\ttfamily unix-\/like}, the class still uses the method {\ttfamily write} of {\ttfamily std\+::ostream} in standard library.
