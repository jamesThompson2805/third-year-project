\doxysubsection{Detailed Description}
A wrapper that stores the base magnitude for unit scaling in formatted output.

Defines the threshold at which values are converted to higher-\/order units (e.\+g. 1000 -\/\texorpdfstring{$>$}{>} "{}1k"{}, 1000000 -\/\texorpdfstring{$>$}{>} "{}1\+M"{}).

The effective range is between 1 and 65535.


\begin{DoxyItemize}
\item A zero value implies no scaling (raw numeric display).
\item Typical usage\+: 1000 (decimal) or 1024 (binary) scaling. 
\end{DoxyItemize}
