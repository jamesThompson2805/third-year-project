 The data consist of "{}pseudo"{} time series obtained by converting two-\/dimensional shapes into one-\/dimensional time series (see \mbox{[}1\mbox{]}, \mbox{[}2\mbox{]}).

There are five classes corresponding to different shapes.


\begin{DoxyItemize}
\item Class 1\+: Arrowhead
\item Class 2\+: Butterfly
\item Class 3\+: Fish
\item Class 4\+: Seashell
\item Class 5\+: Shield
\end{DoxyItemize}

We make two data sets out of these data, Mixed\+Shapes\+Regular\+Train and Mixed\+Shapes\+Small\+Train.
