template<template< typename... > class Node>
struct pgbar::__details::traits::InheritFrom< Node >

A template class that records the inheritance structure of a template class.

To record the inheritance structure of a template class, you need to manually provide a specialized version of the template class type that holds both the VBs and NBs types.
