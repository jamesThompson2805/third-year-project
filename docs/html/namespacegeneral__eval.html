Detailed Description

general_eval is a namespace containing generic methods to compare a series with approximations of itself.

unsigned int num_params,
DRT f,
COMP comp )

comp_of_method compares a series with the uncompressed approximation of itself under the provided distance measure.

Parameters
sis a reference to the series used.
num_paramsis the number of parameters you wish to grant the DRT to approximate with.
fis the DRT to be used (eg. PAA).
compis the comparison function (eg. L2 or even non-norm such as Squared Error).
Returns
The observed distance between the exact series and its approximation.

Seqd general_eval::get_comp_over_num_params ( const Seqd & s,
Sequi vec_params,
DRT f,
COMP comp )

get_comp_over_num_params compares a series with the uncompressed approximation of itself for a variety of sizes under the provided distance measure.

Parameters
sis a reference to the series used.
sizesis a vector of sizes the dataset is to be shortened to. Order doesn't matter as copies of the sequence are made but sizes must be smaller than the original.
num_paramsis the number of parameters you wish to grant the DRT to approximate with.
fis the DRT to be used (eg. PAA).
compis the comparison function (eg. L2 or even non-norm such as Squared Error).
Returns
The observed distances between the exact series (at various lengths) and its approximation of those lengths.
