void plot::barplot_many_series ( const std::vector< Series > & vs,
const std::vector< std::string > & group_labels,
PlotDetails p )

barplot_many_series plots multiple series, here each series is viewed as a value for each set of bars.

Parameters
vsIs the vector of all Series to be plotted. Each series should denote one 'case's set of bars for each group, for example, perhaps you are comparing price of apples and pears bought by Alice, Bob and Charlie. Then passing { {1,3}, {4,3}, {5,6} } indicates Alice bought 1 apple, 3 pears, Bob bought 4 apples, 3 pears and so forth.
group_labelsIs a vector that gives a label to each group which should be of a length equal or greater to that of the inner lists. Using the example beforehand, passing { "apple", "pear" } would be the correct labels for the groups
pIs the PlotDetails.

void plot::plot_lines_generated_ucr_average ( const std::vector< std::string > & dataset_names,
std::string dataset_filepath,
unsigned int ds_size,
std::vector< unsigned int > x,
std::vector< LineGenerator > y_gens,

plot_lines_generated_ucr_average plots all of the lines passed to it using the PlotDetails also passed, utilising multiple UCR datasets.

Parameters
dataset_namesIs the vector of names of the UCR datasets used.
dataset_filepathIs the filepath to the set of the UCR datasets, these can be downloaded from https://www.cs.ucr.edu/%7Eeamonn/time_series_data_2018/.
ds_sizeIs the size the datasets will be set to. This should be small enough such that none are below this size beforehand
xIs the underlying x-axis to plot on (and the values passed to the parameter of LineGenerator).
y_gensIs the vector of LineGenerator that provide method to vary approximation of s.
pIs the PlotDetails to render the graph. Multiple datasets are used via ensuring all are z-normalised and hence being capable to take the mean of all of their results, providing the single line.
