Detailed Description

template<Channel StreamType>
class pgbar::__details::io::OStream< StreamType >

A helper output stream that writes the data to stdout or stderr directly.

It holds a proprietary buffer so that don't have to use the common output buffers in the standard library.

If the local platform is neither Windows nor unix-like, the class still uses the method write of std::ostream in standard library.
