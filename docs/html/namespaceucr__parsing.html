vector< double > ucr_parsing::parse_ucr_dataset ( std::string dataset_name,
std::string dataset_loc,
DatasetType type )

parse_ucr_dataset function takes a datasets name, the location of all UCR datasets and which files to be read and returns the Time Series in the file(s).

Parameters
dataset_nameis the name of the dataset, the name of the folder the dataset is located in and the prefix of the name of the files.
dataset_locis the folder location of the datasets from the present working directory, as it points to a folder, the string should end in /.
typeis the files from the dataset that should be read, options TEST, TRAIN or TEST_APPEND_TRAIN specify the Time Series returned based on TEST and TRAIN present in directory.
Returns
The series read from the file. The function skips any empty or NaN values in the file and skips the first entry of every line as this denotes the 'class' of the line in the UCR Time Series Archive. Files inside the folder of the dataset should be of the format <dataset_name>_TEST.tsv or <dataset_name>_TRAIN.tsv.
